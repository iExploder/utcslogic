# Wagon Data Types
General Data Type
```ocaml
DATA :=
    PM, DS
```
Priminary data types
```ocaml
PM :=
    bool,
    int8, int16, int32, int64,
    float32, float64
```

Data Structures
```ocaml
DS :=
    struct(DATA), array(DATA)
```
Functions
```ocaml
FX :=
    fx(data list, data)
```
